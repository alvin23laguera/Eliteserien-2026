# Eliteserien-2026
Using Match event data from Opta, a review of the present Norweigan Eliteserien, highlighting tactical trends and team profiles, along with individual performance metrics across high impact visualizations. Updated every other matchday. 
Data from every single fixture of the 2026 season, every team and player can be highlighted across different perfomance indicators. This enables the user to see a week-by-week progression.
